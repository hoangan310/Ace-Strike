# Sky Vanguard: Ace Strike

A high-octane, procedurally generated space shooter built with React, TypeScript, and HTML5 Canvas.

## Overview
Sky Vanguard is a top-down "shmup" (shoot 'em up) where players control a customizable starfighter against waves of increasingly difficult enemies and massive bosses. The game features a technical, high-contrast aesthetic with fluid animations and responsive controls.

## Key Features
- **Customizable Ship**: Choose between different hull shapes (Vanguard, Phantom, Titan, Striker) and colors.
- **Dynamic Weapon Systems**: Multiple weapon types including Normal, Spread, Laser, Plasma, Wave, and Homing missiles.
- **Procedural Bosses**: Four unique boss types with distinct attack patterns and visual designs.
- **Difficulty Scaling**: Four difficulty levels (Easy, Normal, Hard, Insane) that affect spawn rates and score multipliers.
- **Power-up System**: Collect upgrades for weapons, health, shields, and tactical drones.
- **Technical UI**: A clean, HUD-driven interface with real-time stats and boss health tracking.

## Tech Stack
- **Framework**: React 18+
- **Language**: TypeScript
- **Rendering**: HTML5 Canvas API
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **Styling**: Tailwind CSS

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Controls
- **WASD / Arrow Keys**: Movement
- **Space / Z**: Primary Fire
- **X**: Fire Missiles
- **Esc**: Pause Game
